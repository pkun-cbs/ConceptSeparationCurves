# ConceptSeparationCurves

This repository contains code and example usage for Concept Separation Curves.

## Getting started

To run the files we advise first installing the packages from the requirements.txt and setup a Python (3.13) environment.
After this setup you can run the following:

```
streamlit run CSC_Streamlit.py
```

This will launch a streamlit webserver and should open a browser window.
In this window you can test, configure, and run the algorithm proposed in our paper.
